# Algorithm

Leetcode 95: [Unique Binary Search Trees II](https://leetcode.com/problems/unique-binary-search-trees-ii/description/)

![algorithm desc](/assets/img/week03-alg.png)

```java
class Solution {
    public int numTrees(int n) {
        int[] G = new int[n + 1];
        G[0] = 1; G[1] = 1;

        for(int i = 2; i <= n; i++){
            for(int j = 1; j <= i; j++){
                G[i] += G[j - 1] * G[i - j];
            }
        }
        return G[n];
    }
}
```
